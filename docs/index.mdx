# What is Mimid?

Mimid is a music player designed for learning songs by ear.
Mimid runs in a browser and is made for mobile devices.
Mimid is free to use, ad-free, and free and open source.

## Features

- Large, easy-to-press buttons.
- Change playback speed with a single tap.
- Loop through difficult sections of a song.
- Precise controls for jumping backward and forward in a song.
- Fully customizable.
- No MP3 files needed.
